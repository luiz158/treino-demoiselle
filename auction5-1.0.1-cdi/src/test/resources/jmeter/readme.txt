This is a readme for a jmeter project StressTest.jmx

To run you have to firt install jakarta-jmeter, then you can run jmeter from /bin directory.
Select the file StressTest.jmx.

For it work firt start the database and the application server.

Inside jmeter you can configure the reports results for a local file ( files can have Gbs...)

for more instruction: http://jakarta.apache.org/jmeter/
